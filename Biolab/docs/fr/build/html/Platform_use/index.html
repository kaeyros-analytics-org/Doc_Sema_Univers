Une fois connecté, vous pouvez accéder aux principales fonctionnalités de BIOLAB.

Rendez-vous

  • Créer un rendez-vous : choisissez une date et un horaire pour un patient.

  • Voir la liste des rendez-vous : affiche tous les rendez-vous déjà planifiés.

Examens

  • Consulter la liste des examens disponibles.

  • Voir les détails d’un examen pour comprendre les informations associées.

  • Créer un nouvel examen si nécessaire.

  • Rechercher un examen dans la liste pour gagner du temps.

Patients

⚠️ Cette section est sensible car elle contient les dossiers médicaux.

  • Une nouvelle authentification peut être demandée pour sécuriser l’accès.

  • Voir le dossier d’un patient pour suivre son historique médical.

  • Créer un patient si un nouveau patient doit être enregistré.

  • Démarrer l’examen d’un patient pour suivre le traitement ou la procédure médicale.

Tableau de bord

  • Afficher un résumé des activités de la plateforme.

  • Montrer des statistiques et indicateurs chiffrés pour suivre l’activité du laboratoire rapidement.
